
# Library

A comprehensive library system that aggregates research data from multiple repositories including E-LIS, DSpace repositories, and global research data sources.

**jobs.vuyo [at] gmail [dot] com**  
(Email obfuscated for security.)

## Features

- **Multi-source harvesting**: 9 DSpace repositories + E-LIS + global research data sources (Dryad, Zenodo, Mendeley)
- **Live E-LIS search**: Real-time searching of Library & Information Science content
- **Automatic daily harvesting**: Scheduled updates at 2 AM every day
- **RIS export**: Export selected records to RIS format for reference managers
- **Advanced filtering**: Filter results by year, repository, type, author, and keywords
- **Responsive UI**: Optimized for desktop, tablet, and mobile devices
