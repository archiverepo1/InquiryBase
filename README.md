# Academic Library Harvester

A comprehensive academic library harvester that aggregates research data from multiple repositories including E-LIS, DSpace repositories, and research data sources.

## Features

- **Multi-source harvesting**: 9 DSpace repositories + E-LIS + research data sources
- **Live E-LIS search**: Real-time searching of library science content
- **Automatic daily harvesting**: Scheduled updates at 2 AM daily
- **RIS export**: Export selected records to RIS format
- **Advanced filtering**: Filter by year, repository, type, and author
- **Responsive design**: Works on desktop and mobile devices
